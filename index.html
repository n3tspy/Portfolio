GLPI plugin tasklists CVE-2024-56801

Vulnérabilité : Blind SQL Injection
Version : < 2.0.4
Criticité : CVSS 9.8 (critical)
Publication : 30 Decembre 2024
Patch : >= 2.0.4
La faille Blind SQLi qui permet d’interoger la base de donnees sans voir le resultat directement. Les reponses de la bases doivent etre deduites par l’attaquant a partir du comportement de l’application (code reponse HTTP, delai de chargement …). Tres puissant pour les attaquants et dangereux pour les sites mais reste beaucoup plus lent a exploiter qu’une injection SQL classique. Pour y remedier il suffit de faire des requetes preparer et parametrer dans le code source et de ne pas concatener du code directement dans la requete SQL.
Patch : Utilisation de la classe DBmysql du framwork GLPI https://github.com/InfotelGLPI/tasklists/commit/6444026e3d2b8fb22d5e5ab03fb86056e1ac9e43 https://fossies.org/linux/glpi/src/DBmysql.php

Source : https://feedly.com/cve/CVE-2024-56801

RCE CURSOR 1.2.4 - CVE-2025-54136

Vulnérabilité : Remote Code Execution (RCE)
Version : <= 1.2.4
Criticité : CVSS 7.2 (high)
Publication : 4 Août 2025
Patch : >= 1.3
La faille MCPoison exploite le mécanisme de confiance des fichiers de configuration MCP (Model Context Protocol) dans l'éditeur Cursor. Lorsqu'un fichier de configuration MCP est présenté pour la première fois, l'utilisateur doit l'approuver manuellement. Cependant, toute modification ultérieure de ce même fichier est automatiquement considérée comme fiable, sans aucune nouvelle validation. Un attaquant ayant accès en écriture à un dépôt GitHub partagé (ou capable d'écrire localement sur la machine cible) peut donc remplacer silencieusement une configuration MCP inoffensive par des commandes malveillantes. Ces commandes s'exécutent automatiquement à chaque ouverture de l'IDE, sans déclencher la moindre alerte. C'est une attaque particulièrement dangereuse car elle cible directement l'environnement de travail du développeur, est persistante (s'exécute à chaque lancement de Cursor), et totalement silencieuse (aucun avertissement affiché). Patch : Mise à jour vers Cursor 1.3 https://github.com/advisories/GHSA-24mc-g4xr-4395

Source : https://nvd.nist.gov/vuln/detail/CVE-2025-54136
